Evaluation Guidelines

  1. There are three levels relevance:
    • 2: highly relevant
    • 1: minimum relevant
    • 0: not relevant
  2. An App should be judged as highly relevant if:
    • it is clearly the app trying to find
    • or, it can satisfy all the functions specified in the query.
  3. An App should be judged as minimum relevant if
    • it can only only provide part of the functions specified in the query
    • or, it is related to the app but a different version
  4. An App should be judged as not relevant if it can not be categorized into the previous two groups
  5. Be strict. If you think an app falls in the border of two categories, choose the lower one
  6. Be consistent. Use the same rule when judge the queries.
